# Animegif
Provides `animegif` command to search animated gifs.
Only works with Mac OS X and Linux.

# Usage

```
$ animegif --help
Usage of animegif:
  -c=8: count
  -k="yuyushiki": keyword

$ animegif -k=ゆゆ式 -c=10 # opens html file with 10 ゆゆ式 gifs.
```

# Installation

```
go get github.com/negipo/animegif
```
